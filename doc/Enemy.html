Class: Enemy

Enemy

new Enemy(difficulty)

The Enemy class controls a single artificially-intelligent enemy. It's a frontend AI class, used for abstracting away all of the backend into a single uniform interface. The implementation of the backend is entirely arbitrary, as long as it exposes a "tick" function which is executed once per logic loop. The backend should also expose a Decision object which is examined each tick and used to control the ship object. So this class can be seen as a sort of "bridge" between the Brain backend and the Ship frontend.
