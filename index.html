  • guide Shows a popup dialogue box listing all the available commands and their descriptions.
  • help command Shows help on the console for the command 'command'.
  • thrust booster power [until condition OR for time] Powers 'booster' to 'power'. If 'until condition' is used, switches 'booster' off when the given condition evaluates as true (for example, "thrust fore.vertical 1 until pitch is 1" turns fore.vertical on until the ship's pitch reaches 1.00). If 'fore time' is used, switches 'booster' off when the given time period has elapsed. Time can be given in seconds, milliseconds, minutes or frames but a unit must be given. Short units (e.g. "m", "s") are supported but be aware that you need to place a space in between the number and the unit.
  • stop booster Turns off 'booster'
  • spawn Spawns a target drone for you to practice against.
  • fire cannon [power] Fires a projectile from 'cannon'. If 'power' is given, the projectile will be fired at that strength. Between 0.0 and 1.0.
  • command name { stuff } Creates a new command named 'name' which will essentially group all of the commands within the brackets to that name. You can also use it to alias longer, more complex commands if you wish. Note that semi-colons are used to separate commands throughout the language, and that includes the 'stuff' here.
  • run command Runs the custom command 'command'. Note that this will not work with the built-in commands.
